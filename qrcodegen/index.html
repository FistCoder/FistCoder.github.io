Générateur de QR code Global

Comment ça fonctionne:

1. Entrez l'URL : Saisissez le lien de l'article JW.ORG dans le champ.
ATTENTION: Trouvez ce lien en bas de n'importe quel article ou video JW.ORG en cliquant sur le bouton Partager. Voir l'image
2. Générez le QR Code : Cliquez sur le bouton pour créer le QR code correspondant à l'URL.
3. Téléchargez le PDF : Après avoir vu le QR code, un autre bouton apparaît pour vous permettre de télécharger un PDF avec le QR code.
C'est un moyen simple de créer et télécharger un QR code en quelques clics.
